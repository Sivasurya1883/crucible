# Incident Commander AI 🚨

An autonomous AI agent that manages production incidents like a real SRE team.

## Features
- Incident triage and severity classification
- Root cause analysis
- Resolution planning with rollback
- Stakeholder communication
- Postmortem generation

## Demo Scenario
Input:
"Payment API returning 500 errors"

Output:
- Severity classification
- Root cause
- Fix plan
- Communication message
- Postmortem

## Note
This demo uses simulated incidents and playbooks but is designed for real-world integration.
